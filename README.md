
# shorten-url-service
단축 URL 서비스 레포지토리


## Project Overview

- bitly와 같은 단축 URL 서비스를 구현한다.
- 사용자가 입력한 원본 URL을 단축 URL로 변환하고 단축 URL 요청 시 원본 URL로 리다이렉트한다.


## Purpose
* REST API를 실제 화면과 연동하며 프론트엔드 요청 및 응답 흐름 이해
* JavaScript(fetch) 기반 비동기 통신과 JSON 직렬화/역직렬화 과정 학습
* DOM 조작을 통해 서버 응답 데이터를 화면에 동적으로 렌더링하는 구조 익힘
* HTML과 CSS를 분리하고 UI 구조를 개선하며 관심사 분리 원칙 적용
* Bootstrap을 도입해 컴포넌트 기반 UI 설계 및 반응형 레이아웃을 학습

## API 
- 단축 URL 생성 API
- 단축 URL 리다이렉트 API
- 단축 URL 정보 조회 API
